SmartBuild is its own web application.
It integrates with Paragon's Paradigm system so design/material outputs can move from blueprint to order quickly.
Manual or automatic material list transfers for entire orders in a single click, along with updating builder pricing multiple ways, this integration can be a huge timesaver.
