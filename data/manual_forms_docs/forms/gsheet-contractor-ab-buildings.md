Contractor Preferences

- GENERAL NOTES
- They get their own OH Doors
- one piece OH Door trim
- bold face J
- 50 Pound ground snow load
- ADD FP16
- If book shelf girts take into account extra blocking material and size of post
- They will want all hardware and brackets needed for a job
- For homeowner quotes we send to A&B to have them bid please send to Nehimiah nconner@abbuildings.com
- NEW STANDARDS
- 8' OC post and trusses- no header system
- 2x6 saddle hangers ( PFD26B) and LU26 for for gables
- probably need drop gables
- cantilever style overhangs
- composite FP
- pair of uplift plates
- special trim fascia
- special trim ridge
- special trim fj
- 213 for gable trim
- 26 gauge on roof from drip edge up
- inside corner with built in J
- 2x8 for PT
- OSV for ridges
- vented soffit for eaves
- H1 for outlooker connection on gable overhangs
- ROUND UP CLOSURES AND OSV FOR RIDGE
- always match the drop gable to the purlins, if it is a 2x6 purlin it needs a 2x6 drop they do not want 2x4 drop
- Account for all wood exposed for liner packs, windows and door sills and behind garage door spring, flat stock material not ribbed
- for open gables use a post type wrap for exposed corners
- one piece GABLE to start at peak then large piece till end
- SEND OVER PLANS EACH TIME
- Rivets to match trim color
- include additional 2x8 board above garage doors to mount spring.
- 2x4x16fr for rat runs even if 8' OC trusses
- call out window height
- send truss seals to forman Ron - ronnjeffery57@gmail.com
- prefer 5/8 OSB for roof even if purlins
- walk door trim when in-swing door
